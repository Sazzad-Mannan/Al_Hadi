# Al-Hadi International Academy Website

A modern, responsive, and visually engaging school website developed for **Al-Hadi International Academy**. The website showcases the school's facilities, academic programs, cultural activities, admission information, teachers, events, notices, and contact details in a professional and user-friendly manner.

## 🌟 Features

* Modern and responsive design
* Attractive homepage slider
* Sticky navigation with active section highlighting
* School facilities showcase
* About Us and Cultural Activities section
* Academic Programs / Classes section
* Teacher showcase slider
* Admission information and downloadable admission form
* Upcoming Events section
* Notice Board section
* Parent testimonials
* Contact form and contact information
* Mobile-friendly layout
* Smooth animations and transitions

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap 5
* jQuery
* WOW.js
* Owl Carousel
* EasySlides Slider
* Font Awesome
* Bootstrap Icons
